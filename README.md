# Wallet

Wallet is an application that simulates the standard use of a wallet.

We only have money in the wallet at the moment.

You can perform the following actions on the wallet:

- 1- Check balance

- 2- Insert money

- 3- Withdraw money

- 4- View your activity history

- 5- Exit
